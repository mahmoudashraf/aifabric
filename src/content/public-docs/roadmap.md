# Roadmap

AI Fabric is in the `0.x` release line. APIs may still change before `1.0.0`, but the current priority is adoption quality over adding another broad module.

## Near-Term Priorities

| Priority | Work |
| --- | --- |
| Public onboarding | Keep docs, examples, and Maven coordinates current |
| Demo reliability | Keep deployed demos resettable, observable, and honest about backend state |
| Provider coverage | Improve Spring AI-backed LLM/embedding support and document escape hatches |
| Security clarity | Strengthen PII, access policy, tenant boundary, and governance examples |
| Testing evidence | Keep CI, manual provider suites, and real-app smoke paths easy to run |
| Community funnel | Maintain starter issues, contributor docs, and small first tasks |

## Not The Current Focus

- Unsupported compliance claims.
- Fake UI-side intelligence.
- New modules without real app evidence.
- Public claims not backed by code, tests, or deployed demos.
