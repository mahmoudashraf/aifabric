# Live Demos

The public demos are real Spring Boot applications backed by AI Fabric modules. They are designed to show the framework through product-shaped scenarios instead of isolated toy endpoints.

| Demo | Capability |
| --- | --- |
| AI Shopping Experience | RAG stages, commerce modes, chat memory, cart and checkout actions |
| AI Fabric Account Resolver | Policy-aware account repair, profile read actions, confirmations, refunds |
| AI Fabric Behavior Signals | Raw event analysis, LLM insight, behavior-driven UI composition |
| AI Fabric Tenant Guard | Tenant-scoped retrieval, role visibility, guarded writes, delete evidence |
| AI Fabric Privacy Shield | PII detection, redacted storage, safe indexing, sanitized search |

## How To Evaluate A Demo

1. Open the demo page.
2. Read the "About this demo" page for backend modules and request flow.
3. Use the suggested scenario.
4. Inspect evidence, actions, confirmation cards, or safety proof.
5. Reset the demo session before trying a conflicting scenario.
